# What's this?

This is a testbed for testing the TCP CCA behaviour based on iperf3.

**NOTE: To run this testbed on receiver side.**

## Dependencies

* python3
* matplotlib
* iperf3

## How to use?

* setup the netem, run `./setup_netem.sh`
* simply begin the test by running `./run.sh`
